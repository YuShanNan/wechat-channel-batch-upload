# 视频号批量上传

一个 Windows 桌面工具，帮你批量往微信视频号上发视频。

## 功能

| 模块 | 说明 |
|------|------|
| **扫码登录** | 点"添加账号"弹出二维码，手机扫一下即可登录。支持多策略二维码查找，适配微信页面改版 |
| **批量上传** | 拖拽视频到队列，填好标题，点"发表"自动逐个上传。支持定时发布、间隔等待 |
| **多账号** | 可添加多个视频号账号，各自独立 profile，互不影响。支持账号有效性检测 |
| **封面复用** | 拖一张封面图，整个队列的视频共用 |
| **进度条** | 按单视频上传进度平滑推进，不是按视频数跳变 |
| **失败重试** | 上传失败后立即重试，最多 3 次，日志和状态同步显示 |
| **完成通知** | 全部上传完成后弹出 Windows 通知气泡 + 系统提示音（可关） |
| **系统托盘** | 关闭窗口缩小到托盘，右键恢复/退出。有任务时退出会确认 |
| **有头模式** | debug 模式下使用系统 Chrome/Edge 浏览器，可观察上传过程 |
| **深色主题** | 支持亮色/暗色切换 |

## 怎么用

1. 下载 `视频号上传.exe`，双击打开
2. 点"管理" → "添加账号"，用微信扫二维码登录
3. 把视频文件拖到左边区域，填一下标题和描述
4. 选好账号，点"发表"

## 自己打包

```bash
pip install -r requirements.txt
python -m playwright install chromium
python app.py          # 开发模式
python -m PyInstaller 视频号上传.spec  # 打包
```

## 注意

- 仅支持 Windows 10/11（需要 Edge WebView2，系统自带）
- 有头模式需要系统已安装 Chrome 或 Edge
- 别拿来发违规内容，封号不关我事
