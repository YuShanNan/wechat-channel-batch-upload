# 视频号批量上传

一个桌面工具，帮你批量往微信视频号上发视频。

## 这玩意儿能干嘛

- **扫码登录**：点一下弹出二维码，手机扫一下就登上了，不用每次打开浏览器
- **批量上传**：把视频拖进去，填好标题，点"发表"，它会自动一个个上传。不用守着网页一个个点
- **多账号**：可以添加多个视频号账号，各自独立，互不影响
- **定时发**：设个时间，到了自动开始发；还能设置视频之间隔多久发一个
- **封面复用**：拖一张封面图，整个队列的视频全用它
- **进度条**：上传时能看到实时进度，不用瞎等

## 怎么用

1. 下载 `视频号上传.exe`，双击打开
2. 点"管理" → "添加账号"，用微信扫二维码登录
3. 把视频文件拖到左边区域，填一下标题
4. 选好账号，点"发表"

就这四步。

## 自己打包

如果你懂一点 Python：

```bash
pip install -r requirements.txt
python -m playwright install chromium
python app.py          # 开发模式直接跑
python -m PyInstaller 视频号上传.spec  # 打包成 EXE
```

## 注意

- 仅支持 Windows 10/11（需要 Edge WebView2，系统自带不用装）
- 上传速度取决于你本地网络和视频大小
- 别拿来发违规内容，封号不关我事
